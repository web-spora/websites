# mb_detect

Лендинг для проверки оригинальности моторных масел Mercedes-Benz.

## Структура

```
mb_detect/
├── index.html          # Главная страница
├── css/
│   └── styles.css      # Стили
├── js/
│   └── main.js         # Скрипты
├── assets/
│   ├── images/         # Изображения
│   └── fonts/          # Шрифты
├── pages/              # Дополнительные страницы
└── agents.md           # Этот файл
```

## Разработка

Открыть `index.html` в браузере через Live Server или напрямую.

## Соглашения

- **CSS** — переменные в `:root`, mobile-first адаптив, классы в kebab-case
- **Изображения** — пути `assets/images/`, атрибут `loading="lazy"`
- **Секции** — обёртка `.section > .container`, заголовок `.section-title`
- **Сетки** — через `grid`, 2 колонки на десктопе → 1 на мобильных
- **Шрифт** — Inter (Google Fonts)

## Сборка

Простой статический сайт, сборка не требуется.

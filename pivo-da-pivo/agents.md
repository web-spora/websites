# AGENTS.md — Пиво-да-Пиво

## Проект
Одностраничный сайт-витрина для магазина крафтового пива «Пиво-да-Пиво» (г. Ульяновск).

## Стек
- HTML5 + CSS3 (Flexbox/Grid)
- Ванильный JavaScript (без фреймворков)
- Шрифт: Shafarik (OTF) в `fonts/Shafarik/`
- Фавикон: `favicon.svg` со стилизованным «Да»

## Дизайн-система
- **Фон:** #0d0d0d (глубокий чёрный)
- **Карточки:** #1a1a1a
- **Акцент:** #D4AF37 (золото)
- **Текст:** #e0e0e0 / #888
- **Шрифт лого/заголовков:** Shafarik
- **Стиль:** Премиальный, крафтовый, тёмная тема

## Изображения
- Все файлы в `images/` переименованы в латиницу
- Иллюстрации (`illustration-1..5`, `beer-and-pizza`) используются как декор
- Фото блюд и напитков маппятся через `IMAGE_MAP` в JS
- Для позиций без фото — эмодзи-заглушки категорий

## Контент
- Hero с заголовком и якорем на меню, фоновые иллюстрации
- Секция "Меню" с табами для категорий: Все, Пиво, Безалкогольные напитки, Пицца, Закуски
- Данные меню в JS-массиве `MENU` (спарсены из `menupivo - Общее.csv`)
- У пицц отображается цена за 30 см и 40 см
- Галерея иллюстраций между меню и «О нас»
- Детальное модальное окно при клике на карточку
- Модальное окно бронирования при клике "Забронировать"
- Секция "О нас" и контакты
- Footer с 18+ дисклеймером

## Источник данных
- `menupivo - Общее.csv` — мастер-файл с полным меню
- При обновлении CSV нужно вручную синхронизировать массив `MENU` в `index.html`

## Правила
- Кнопка действия — только "Забронировать" (не "Купить")
- Никакой онлайн-оплаты и корзины
- Адаптивность Mobile First
